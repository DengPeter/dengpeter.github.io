vagrant

0. vagrant简介及机制介绍

vagrant机制，vagrant采取box和实例的概念，box是专门存放刚导入的镜像box，实例是从box中实例化出来的虚拟机。实际运行的时候是实例

1. 下载安装

2. 使用

  • 增加box
  $ vargrant box  add <boxname>  <path or url>

实例

  $ vagrant box  add nagios /data/iso/ztgt_dev_20160412.box
  • 初始化 启动box 登录box
  $ vagrant init
  $ vim VagrantFile
    config.vm.box = "nagios" #指定box
    config.vm.network "private_network", ip: "192.168.33.10"

  $ vagrant up     #启动虚拟机
  $ vagrant ssh    #登录虚拟机

销毁实例: 销毁当前正在运行的实例

   $ vagrant destroy
