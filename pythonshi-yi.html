global

error: SyntaxWarning: name ‘xxxxxxxxxxx’ is assigned to before global declaration 此错误是因为变量赋值之后，再global 变量名。正确用法是先global 变量名 再给变量赋值

def a():
    x = 'sdf'
    global x    #这会导致SyntaxWarning

def b():
    global x
    x = 'sdf'   #此列为正确用法
