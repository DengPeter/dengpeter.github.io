前端问题汇总

1. 改变字体大小 位置的变化

html code:

<nav class="top-nav">
  <h1 class="ml10 linehgt40 font16">System</h1>
  <ul class="font16" >
    <li'>'</li>
    <li'>'</li>
  </ul>
</nav>

css code:

html,body{margin:0;pading:0;width:100%;height:100%}
.top-nav{
  background-color: #141414;
  height: 40px;
  color: #fff;
}
.top-nav h1{
  position: absolute;
  left: 20px;
  top: -10;
}
.top-nav ul{
  position: absolute;
  left:80px;
  top: -5;
}

问题描述： 字体的垂直位置是由绝对路径absolute top定位的，一旦字体大小改变top值也要跟着改变。有什么方法可以固定h1 和 li的位置，改变字体大小也不影响位置？ 解决方案：

暂无

2 字体颜色大小等样式，只能用在包裹文字的标签上，在包裹文字标签的父级标签不生效

body 背景不能顶头

html

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <link rel="stylesheet" href="/static/css/base.css">
  <title>Document</title>
</head>
<body>
  <div class="container">
    <nav>
      <h1>Sytem</h1>
      <ul>
      <li'>'</li>
      <li'>'</li>
    </ul>
    </nav>
  </div>
</body>
</html>

css

html{
  background-color: #cccccc;
  margin: 0;
  padding: 0;
  height: 100%;
  width: 100%;
}

body{
  margin: 0;
  padding: 0;
  background-color: #7f7f7f;
  height: 100%;
}

3 line-height的用法 和 height的区别

在使用 height: 40px;显示的是 40px高度， 但是使用了line-height: 40px; 的时候实际显示为200px; 原因： line-height不能用在父标签上，而应该用在子标签上 html code:

<div>
  <ul>
    <li'>'</li>
  </ul>
</div>

css code:

div{
  height: 50px;
}
a{
  line-height: 50px;
}
