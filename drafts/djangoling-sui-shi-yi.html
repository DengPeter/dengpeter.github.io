在子模版中引入css

  1. 在父模版和子模版顶部写: {% load staticfiles %}
  2. 在父模板头部link下面写：{% block extracss %}{% endblock %}
  3. 在子模板中引入css {% block extracss %} <link rel="stylesheet" href="/static/css/crm/login.css"/> {% endblock %}
