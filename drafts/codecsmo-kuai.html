codecs模块

编码转换总是麻烦的，python中的编码转换却非常的方便

decode –> unicode –> encode 原有编码 –> 内部编码 –> 目的编码

python的内部是使用unicode来处理的，但是unicode的使用需要考虑的是它的编码格式有两种，一是UCS-2，它一共有65536个码 位，另一种是UCS-4，它有2147483648g个码位。对于这两种格式，python都是支持的，这个是在编译时通过–enable- unicode=ucs2或–enable-unicode=ucs4来指定的。那么我们自己默认安装的python有的什么编码怎么来确定呢？有一个 办法，就是通过sys.maxunicode的值来判断:

``` import sys print sys.maxunicode

```

  1. 将任何字符编码转换到 unicode 用decode
  2. 将unicode转换到任意字符编码 用encode

open 和 codecs.open 的区别

对于open文件的操作，通常是 1. read file —> 2. 操作内容 —> 3.保存内容 对应编码转换为： 读 转换 1. read file –> decode 内容到 unicode 转换 存 2. encode 内容到utf8 –> save file

f=open(‘name.file’, ‘r’) content=f.read().decode(‘utf-8’)

codecs库读的过程自动转换 f=codecs.open(‘name.file’, encoding=’utf-8’) content=f.read()
