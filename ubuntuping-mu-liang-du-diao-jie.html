ubuntu屏幕亮度调节

adjust screen brightness 屏幕亮度调节

买回来的新笔记本装上Ubuntu16.04发现fn不能调节屏幕亮度，google之发现是显卡是inter集成核显非nvidia。 fn对应管理的文件不存在。 写个脚本调整屏幕亮度,然后自定义一个系统快捷键即可方便的调整屏幕亮度了。

增减屏幕亮度

#!/bin/bash
# descritption: 改变屏幕亮度 增加亮度

 lightfile=/sys/class/backlight/intel_backlight/brightness
 req=50
 lightcount=`cat $lightfile`

 if [ $lightcount -lt 937 ];then
    echo 'passwd'| sudo -S ls . >> /dev/null #echo 密码给sudo
    echo `expr $lightcount + $req`  |sudo tee  /sys/class/backlight/intel_backlight/brightness >>/dev/null
 fi

降低亮度

 #!/bin/bash
 #descritption: 改变屏幕亮度 增加亮度

 lightfile=/sys/class/backlight/intel_backlight/brightness
 req=50
 lightcount=`cat $lightfile`
 if [ $lightcount -gt 0 ];then
    echo 'passwd'| sudo -S ls . >> /dev/null  # 将密码输入给sudo 下面调用sudo的时候就不用输入密码了
    echo `expr $lightcount - $req`  |sudo tee  /sys/class/backlight/intel_backlight/brightness >>/dev/null
 fi
