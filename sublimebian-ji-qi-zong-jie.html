快速切换工作目录：

SideBarFolders Folders增强工具 sublime的键盘控制，没有vim那么强大但用触摸板的还算比较流畅 folders 直接append日常需要使用的工作目录，个人觉得还省去了project的管理麻烦

命令行中快速打开sublime

mac中安装的软件通常在命令行中都找不到命令，是因为系统命令变量中没有命令路径 我们可以自己软链接或者直接加入系统变量

以sublime为例： ln -s /Applications/Sublime Text.app/Contents/SharedSupport/bin/subl ~/bin/sublime

打开方式 sublime

open directory

sublime /data

open file

sublime text.html

sublime快捷键

ctrl+k ctrl+b #打开隐藏侧边栏 同时按下2个快捷键（先c+k,b)

启用Package Control

ctrl+` 调出命令行运行下面代码sublime3

import urllib.request,os; pf = ‘Package Control.sublime-package’; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); open(os.path.join(ipp, pf), ‘wb’).write(urllib.request.urlopen( ‘http://sublime.wbond.net/’ + pf.replace(‘ ‘,’%20’)).read())

SublimeCodeIntel插件

智能提示插件，这个插件的智能提示功能非常强大，可以自定义提示的内容库 Preferences ——》Package Settings ——》SublimeCodeIntel ——》Settings - User 修改里面python的配置，如下：

"codeintel_config": {
    "Python": {
            "PATH": "usr/bin:/bin:/usr/sbin:/sbin:/usr/local/bin:$PATH",
            "pythonExtraPaths": ["/usr/local/lib/python2.7/site-packages"],
            "python":"/usr/bin/python2.7",
    },
}

前端补全 emmet插件

emmet插件默认是用 ctrl+e补全，但这在*unix平台跟系统默认的ctrl+e 跳转光标到行尾冲突。 修改快捷键 KeyBindings - User 中配置：

[
{ "keys": ["ctrl+e"], "command":"move_to", "args":{"to":"eol"}},
{"keys": ["ctrl+d"], "args": {"action": "expand_abbreviation"}, "command": "run_emmet_action", "context": [{"key": "emmet_action_enabled.expand_abbreviation"} ] },
]

主题皮肤 Brogrammer

  1. 下载 https://github.com/kenwheeler/brogrammer-theme/archive/master.zip
  2. 解压并重命名为 Theme - Brogrammer
  3. 复制到用户package目录中 ｀Preferences -> Browse Packages｀
  4. 启用皮肤 编辑用户配置文件 ｀Preferences -> Settings - User｀ 然后重启编辑器
{
  "theme": "Brogrammer.sublime-theme",
  "color_scheme": "Packages/Theme - Brogrammer/brogrammer.tmTheme"
}

5.个性化 看起来标题栏比较高,把它变矮点 配置文件在：｀Preferences -> Browse Packages｀下面的｀Theme - Brogrammer/Brogrammer/Brogrammer.sublime-theme｀ 用编辑器打开 Brogrammer.sublime-theme

标题栏高度在

// TABS (REGULAR)
    1. 修改： "tab_height": 50,   这个值 我自己改为30
    标题变矮了可是，背景栏还是那么高碍眼继续改，花了好久才发现整个背景栏并非是用一个参数设定的，而是一段代码块 TAB LABELS 设定的。把它全部注释158行到249行。 这下标题栏就算搞定。
    ／*
    // TAB LABELS
    ....
    *／

安装anaconda后 打开python文件出现白框

原因：代码不符合PEP8格式要求 导致自动补全的anaconda插件提示框框

解决办法：｀Sublime>Perferences>Package Settings>Anaconda>Setting User｀ 增加一句配置 ｀{“anaconda_linting”:false}｀
