{{ 变量 }}返回安全字符

服务器端返回了html代码，默认django出于安全考虑只当成字符串显示并不会将代码渲染,django在模板引擎中提供了autoescape来渲染。 服务器端：

data = "<h1> hello </h1>"

前段模板:

{{ data }}

上面的变量data 并不会被渲染为h1标题，而是原样输出"<h1> hello </h1>"

{% autoescape %} {{ data }} {% endautoescape %}

子模板引入css文件

html文件: base.html 和login.html css文件: base.css 和login.css 关系： 1. base.html引入base.css 2. login.html 继承base.html 3. login.css 中引入login.css

code： base.html

{% block exracss %} {% endblock %}

login.html

{% load staticiles %}
{% block exracss %}
<link rel="stylesheet" href="/static/css/login.css" media="screen" title="no title" charset="utf-8">
{% endblock %}

子模板中：{% load staticfiles %}要写在第二行

静态文件变量

在模板中静态文件引用时使用变量，方便setting中的修改。

原先硬编码的写法: <link rel="stylesheet" href="/static/css/reset.css"/>

使用变量的方法

{% load staticfiles %}
{% block extracss %}
<link rel="stylesheet" href="{% static "css/reset.css" %}"/>
<link rel="stylesheet" href="{% static "css/navigation.css" %}"/>
<link rel="stylesheet" href="{% static "css/register.css" %}"/>
