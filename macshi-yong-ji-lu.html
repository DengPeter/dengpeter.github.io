终端中打开窗口

$ open .

窗口中打开终端:

1. 在键盘快捷键中设置(勾选 加快捷键设置)

    系统偏好设置>键盘>服务>文件和文件夹>新建于文件夹位置的中断标签

2. 在finder中打开
然后在Finder右键一个文件夹，右键菜单里面 Services 选择 New Terminal Tab at Folder 就打开了当前位置的终端了。类似Linux下面的右键，用终端打开。

启动ssh

#1.编辑/etc/sshd_config文件，注释掉
#ForceCommand /usr/local/bin/ssh_session

#2.启动sshd服务：
sudo launchctl load -w /System/Library/LaunchDaemons/ssh.plist

#3.停止sshd服务：
sudo launchctl unload -w /System/Library/LaunchDaemons/ssh.plist

#4查看是否启动：
sudo launchctl list | grep ssh

如果看到下面的输出表示成功启动了：
－－－－－－－－－－－－－－
- 0 com.openssh.sshd

#5.最后ssh localhost 成功

brew 命令

brew install  softname
brew uninstall softname
brew search   softname #支持正则
brew list           #列出已安装的软件
brew update         #更新brew
brew home           #用浏览器打开brew的官方网站
brew info           #显示软件信息
brew deps           #显示包依赖

用户管理

创建用户 "opera":
dscl . -create /Users/opera
dscl . -create /Users/opera UserShell /bin/bash
dscl . -create /Users/opera RealName "Lucius Q. User"
# 注意 UniqueID必须唯一
dscl . -create /Users/opera UniqueID "1010"
dscl . -create /Users/opera PrimaryGroupID 80
dscl . -create /Users/opera NFSHomeDirectory /Users/luser

删除用户
dscl . -delete /users/username

修改密码:
dscl . -passwd /Users/opera password

加入admin用户组
dscl . -append /Groups/admin GroupMembership opera

ls 像gnu一样高亮；

Mac OS X 是基于 FreeBSD 的，所以一些工具 ls, top 等都是 BSD 那一套，ls 不是 GNU ls，所以即使 Terminal/iTerm2 配置了颜色，但是在 Mac 上敲入 ls 命令也不会显示高亮，可以通过安装 coreutils 来解决（brew install coreutils），不过如果对 ls 颜色不挑剔的话有个简单办法就是在 .bash_profile 里输出 CLICOLOR=1：

$ vi ~/.bash_profile
export CLICOLOR=1

brew install coreutils

mac 下dd 命令刻录系统到u盘

在linux下直接dd if of就可以开始刻录了，但是在mac里面却提示

sudo dd if=/data/iso/linuxmint-18-cinnamon-64bit.iso of=/dev/disk2
dd: /dev/disk2: Resource busy

如果直接把磁盘退出又完全退出了，找了下发现diskutil命令可以直接取消挂载目录而不退出u盘

sudo diskutil unmountDisk /dev/disk2
Unmount of all volumes on disk2 was successfu

取消挂载点后就可以正常dd了
