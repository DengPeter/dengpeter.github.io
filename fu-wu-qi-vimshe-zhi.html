服务器vim设置

虽然我们的工作平台上vim有很好的定制，但还是避免不了很多时候在服务器上vim编辑文件。默认的vim设置编辑配置文件或者代码都是’非常不友好的’。其实只要稍作修改然后在服务器初始化的时候加入进去就可以很好的使用来 (^_^).

在 /etc/vimrc 的最后面加入：

  • 1.tab转空格设置
" ==== custome settings  ======
"1.tab转空格设置

set ts=4 "tabstop
set expandtab "空格作为缩进
set sw=4 "shiftwidth
set autoindent

"行号／相对行号"
set number
set relativenumver

" ==== custome settings  end ======
