stat命令是用来显示文件或文件系统的状态。

$stat cron-20191209
  File: ‘cron-20191209’
  Size: 5546151    Blocks: 10848      IO Block: 4096   regular file
Device: 803h/2051d Inode: 526901      Links: 1
Access: (0640/-rw-r-----)  Uid: (    0/    root)   Gid: (    4/     adm)
Access: 2019-12-01 03:38:01.747042271 +0800
Modify: 2019-12-09 03:36:01.043272358 +0800
Change: 2019-12-09 03:36:01.089272077 +0800
 Birth: -

时间解析

access time：表示最后一次访问（仅仅是访问，没有改动）文件的时间 modify time：表示最后一次修改文件的时间 change time：表示最后一次对文件属性改变的时间，包括权限，大小，属性等等

相关命令对时间的影响

ls 看到的时间是modify时间

cat 命令会更新access time

vim 会更新modify 时间

chmod/chown命令会更新 change时间

touch 会把三个时间都更新
