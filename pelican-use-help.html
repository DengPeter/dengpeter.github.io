部署到github上面

1. 在github.com上面创建一个与自己用户名相同前缀的仓库后缀用github.io

    peterdeng.github.io

2. 下载仓库/上传测试页面

git clone https://github.com/peterdegn/peterdeng.github.io
cd peterdeng.github.io
echo "Hello World" > index.html
git add .
git commit -m 'add a index.html page  as test'
git push origin master
3. 测试

浏览器中访问 peterdeng.github.io

4. 域名解析

4.1 在仓库中增加CNAME文件,写入顶级域名,提交到远程仓库

echo 'pyshell.cn' >CNAME
git add CNAME
git commit -m 'add pyshell.cn to CNAME'
git push origin master

4.2 解析域名

到域名管理后台解析域名，添加一条cname记录到peterdeng.github.io cname @ peterdeng.github.io

4.3 生效后就可以访问新域名了pyshell.cn

2 bug和使用功能需要完善等信息记录

  • pelican 中的markdown链接跳转方式,目前貌似只能在页内，留待后面处理。

解决方法 参考 markdown中的超链接跳转是当前窗口，如果想让它新标签窗口打开，可以用以下代码实现， 原理很简单：遍历\<a href=”#”>标签然后给它加上target=”_blank”属性。

<script type="text/javascript">
    $(document).ready(function() {
        //为超链接加上target='_blank'属性
        $('a[href^="http"]').each(function() {
            $(this).attr('target', '_blank');
        });
    });

</script>
