nagos4.2搭建及使用

nagios搭建篇

系统：centos6.4

依赖清单：

sudo yum -y install wget gcc glibc glibc-common gd gd-devel make net-snmp unzip

http php mysql 之前已经安装这里略过

soft清单：

$ wget http://prdownloads.sourceforge.net/sourceforge/nagios/nagios-4.2.0.tar.gz
$ wget http://nagios-plugins.org/download/nagios-plugins-2.1.2.tar.gz

创建nagios用户

sudo useradd nagios -s /sbin/nologin -M
sudo groupadd nagcmd
sudo usermod -a -G nagcmd nagios

install nagios

$ tar zxvf nagios-4.2.0.tar.gz
$ tar zxvf nagios-plugins-2.1.2.tar.gz
$ cd nagios-4.2.0
$ ./configure --prefix=/opt/nagios --with-command-group=nagcmd
$ make all
$ sudo  make install
$ sudo make install-init
$ sudo make install-config
$ sudo make install-commandmode
$ sudo make install-webconf

$ sudo cp -R contrib/eventhandlers/ /opt/nagios/libexec/
$ sudo chown -R nagios:nagios /opt/nagios/libexec/eventhandlers
$ sudo /opt/nagios/bin/nagios -v /opt/nagios/etc/nagios.cfg

启动nagios,apache

sudo /etc/init.d/nagios start
$ sudo /opt/apache/bin/apachectl start

登录用户名

 $ htpasswd –c /usr/local/nagios/etc/htpasswd.users nagiosadmin
