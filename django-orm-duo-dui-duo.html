django orm 多对多

数据模型： 书 出版社 作者

关系： 书 和 作者 多对多（一本书可以有多个作者，一个作者也可以出多本书） #manyTomany 书和出版社 一对多（一本书只能有一个出版社） #foreignkey

数据库关系： 自关联:

class Comment(models.Model):
  comment_content = models.CahrField(max_length=255)
  parent_comment = models.ForeignKey('self','id',null=True)  # 关联自己

一对多ForeignKey： 一本书只能属于一个出版社

class Published(models.Model): #一个出版社可以有多本书
  pass

class Book(models.Model):
  published = models.ForeignKey(published) # 一本书只能属于一个出版社

基础模型

class Auth(Model):
    lastname = models.Field.CharField(max_length=46)
    firstname = models.Field.CharField(max_length=46)

class Published(Model):
    pub_name = models.Field.CharField(max_length=48)
    pub_address = models.Field.CharField(max_length=128)

class Books(Model):
    bookName = models.Field.CharField(max_length=68)
    auth = models.ManyToManyField(Auth)
    published = models.ForeignKey(Published)

增删该查基础api使用

以数据库的增删改查来对于django orm 的对象api操作。

增/创建对象

auth = Auth.objects.create(firstname="peter",lastname="Deng")
auth.save() # 执行了save方法后对象才把内容写入到数据库

改/修改对象

auth = Auth.objects.get(firstname="peter",lastname="Deng")
auth.firtname = "Sophie"
auth.save()
