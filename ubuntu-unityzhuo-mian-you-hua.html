ubuntu-unity桌面优化

** 问题： ubuntu16.04默认alt键绑定了2个功能：1.显示菜单，2.触发HUD 实际使用中alt-d 是出现HUD搜索框，而非定位到浏览器地址栏。 很多程序窗口有失去焦点时隐藏的功能，但有很多快捷键是alt开始的，当按下alt时显示菜单焦点就失去了。 使用ccsm来取消设置

$ sudo apt-get install compizconfig-settings-manager
$ ccsm

取消HUD:

进入ubuntu unity plugin -> 点击 Key to the HUD when tapped 按钮取消勾选Enable -> 点击OK确定。

取消alt显示菜单

进入 ubuntu unity plugin -> 点击 Key to show the menu bar hwile pressed 将alt键 换为 super键(视窗键)

@ 注：使用ccsm 修改Ubuntu桌面环境，容易造成桌面崩溃。切记不清楚的选项不要修改。

** 问题ubuntu的super键（视窗键）是默认触发启动器的用起来非常不爽，每次按super键的时候都要显示侧边栏和启动器搜索框，并且少了一个非常顺手的功能键，键盘操作效率大打折扣。 改变super的绑定使用 unity-tweak-tools就可以。

$ sudo apt-get install unity-tweak-tools
$ unity-tweak-tools

在 unity -> 附加 -> 显示启动器 -> 鼠标单击super 然后按 backspace 就disable它了。 要再显示启动器和侧边栏的时候使用super + tab 组合键来完成。
