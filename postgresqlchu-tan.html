postgresql初探

第一次使用postgresql，系统为ubuntu。

postgresql是什么？

关系型数据库，由加州大学伯克利分校计算机系开发，以BSD许可开源。

安装postgresql

  • 客户端安装: sudo apt-get install postgresql-client

  • 安装服务端: sudo apt-get install postgresql

命令使用：

登录数据库: sudo -u atausername&systemusername psql -h ip -U username -d dbname
