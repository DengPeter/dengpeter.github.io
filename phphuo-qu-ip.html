php获取ip

最近跟开发联调php获取ip的问题，记录下过程。

首先需求是 要获取1.服务器端IP，2.获取客户端IP

涉及到的几个函数：

gethostbyname($_SERVER['SERVER_NAME']) // 获取到服务器端的公网ip
get_client_ip() //获取客户端的本机ip(thinkphp框架提供)
$_SERVER['REMOTE_ADDR'] //同上
$_SERVER['SERVER_ADDR'] //获取的是运行代码机器的ip  问题：有时候获取的是127.0.0.1

问题1：获取客户端的服务器ip不正确，仔细查找发现原来是跟代理有关

$_SERVER[‘REMOTE_ADDR’获取客户端本机的ip，但是如果客户端有代理的话，得到的ip地址非客户端真实ip，而是代理的ip，此时用 那么用$_SERVER[“HTTP_X_FORWARDED_FOR”]获取，同时也要注意：如果没有设置代理$_SERVER[“HTTP_X_FORWARDED_FOR”] 将得到null空值。可以使用if来判断一下。

<?php
if ($_SERVER["HTTP_X_FORWARDED_FOR"]=="")
{
$user_ip=$_SERVER["REMOTE_ADDR"];
}
else
$user_ip=$_SERVER["HTTP_X_FORWARDED_FOR"];
?>

在thinkphp框架中 get_client_ip()这个函数已经处理了上面的问题，如遇到代理则添加2个参数. get_client_ip(0,true)

问题2：获取本机ip出错，居然是127.0.0.1导致逻辑不能正常跑通。 $_SERVER[‘SERVER_ADDR’] 获取到的ip非运行代码机器的ip，而是127.0.0.1

解决：使用这个就可以

gethostbyname($_SERVER[‘SERVER_NAME’]) /
