tmux使用

场景：在服务器上长时间连续几天的处理问题，遇到网络不问题，下班回家等，再次连接到服务器上要恢复工作环境，打开的文件，日志，运行的程序都得从头来很是麻烦。有了tmux后再也不用为此烦恼了 ^_^! 当在跟远程传输一个大文件，下班的时候还没完成需要回家再继续搞。合上电脑，回到家后发现链接已经断开，好懊恼又要重新连。用上tmux到家只要打开tmux会话一切继续。so easy！ tmux以键盘为主，虽然大部分操作都有快捷键可用，但palne切换，改变palne大小，复制等操作却并非高效，每个操作都至少要按3个键(c-b x )操作成本太高。还好tmux的配置对鼠标支持也非常好。笔记本用上触摸板效率一下就提升上来了。

基本使用：

安装：

yum install tmux

配置文件：

```
全局 /etc/tmux.conf
用户级别: $HOME/.tmux.conf  #优先级高
```

概念：

 ```
  1. session
    在终端可以创建多个session
  2. windows
    在一个session中可以创建多个windows
  3. palne
    在一个window中可以创建多个palne
    刚开是接触tmux会有点不适应它的工作方式，没关系只要多熟悉即可。
```

session 使用：

session的操作基本都是在terminal中操作，跟普通linux命令一致

创建：

```
tmux new -s [sessionName]
tmux new -s local # 创建一个新session 命名为 local
tmux new -s remote # 创建一个session 命名为 remote
tmux new -s test -d #创建一个新session 命名为 test 放在后台运行 #-d在后台运行
```

查看session:

```
 tmux ls

 ```

进入某个session

```
tmux attach -t local  #进入local会话
```

隐藏会话:

ctrl - d #将会话放入到后台

切换session：

c-b s [jk] #先按下 prefix(ctrl-b) 再按s 出现session列表时 用jk上下选择

prefix key 映射

默认的prefix key 是ctrl-b，一个是因为建成比较远，二是因为ctrl-b 在terminal中是跳转到行尾。所以会把它重新绑定。 个人习惯把alt键(非mac键盘)作为meta键，不跟ctrl键的功能冲突。配置文件ctrl键 为大写C，alt键大写M

在配置文件中加入： unbind C-b ＃取消绑定 set -g prefix M-a ＃绑定alt+a键作为prefix键

快捷键

  system:
    prefix - ? #列出所有快捷键；按q返回
             d  脱离当前会话；这样可以暂时返回Shell界面，输入tmux attach能够重新进入之前的会话
             D  选择要脱离的会话；在同时开启了多个会话时使用
             Ctrl+z 挂起当前会话
             r  强制重绘未脱离的会话
             s  选择并切换会话；在同时开启了多个会话时使用
             :  进入命令行模式；此时可以输入支持的命令，例如kill-server可以关闭服务器
             [  进入复制模式；此时的操作与vi/emacs相同，按q/Esc退出
             ~  列出提示信息缓存；其中包含了之前tmux返回的各种提示信息

  session:
    prefix - s # session列表

  window:
    prefix - w  #window 列表
             c  创建新窗口
             &  关闭当前窗口
             1-9    数字切换至指定窗口
             p  切换至上一窗口
             n  切换至下一窗口
             l  在前后两个窗口间互相切换
             w  通过窗口列表切换窗口
             ,  重命名当前窗口；这样便于识别
             .  修改当前窗口编号；相当于窗口重新排序
             f  在所有窗口中查找指定文本

  palne:
    prefix - ”  将当前面板平分为上下两块
             %  将当前面板平分为左右两块
             x  关闭当前面板
             !  将当前面板置于新窗口；即新建一个窗口，其中仅包含当前面板
             Ctrl+方向键   以1个单元格为单位移动边缘以调整当前面板大小
             Alt+方向键    以5个单元格为单位移动边缘以调整当前面板大小
             Space  在预置的面板布局中循环切换；依次包括even-horizontal、even-vertical 、 main-horizontal、main-vertical、tiled
             q  显示面板编号
             o  在当前窗口中选择下一面板
             方向键    移动光标以选择面板
             {  向前置换当前面板
             }  向后置换当前面板
             Alt+o  逆时针旋转当前窗口的面板
             Ctrl+o 顺时针旋转当前窗口的面板
