centos6安装多版本python环境

centos6系自带的python版本为2.6.*,但实际项目中常用2.7和3.4的版本。花了点时间整理了个多版本的环境，由于是在线上服务器出于安全和统一管理没有使用流行的版本管理工具pyenv而是采取直接编译的形式。

0. 前提：

python标准库中http中会用到https，所以确保后期可以正常使用先把openssl,openssl-devel安装上。

sudo yum install openssl openssl-devel

如果要用sqlite则需在安装python前安装sqlite

sudo yum -y install sqlite sqlite-devel

1. 安装不同版本python

2.7后的版本安装过程都是一样的，所以这里以3.4版本为例。需要下载的安装包： - python包 - setuptool包 - pip包 注意各setuptool的版本需要和python版本对应起来。

下载安装包
wget -c  -O setuptools-26.1.1.tar.gz https://github.com/pypa/setuptools/archive/v26.1.1.tar.gz
wget -c https://pypi.python.org/packages/e7/a8/7556133689add8d1a54c0b14aeff0acb03c64707ce100ecd53934da1aa13/pip-8.1.2.tar.gz
wget -c https://www.python.org/ftp/python/3.4.0/Python-3.4.0.tar.xz
安装python3.4
tar axf Python-3.4.0.tar.xz
cd Python-3.4.0
./configure --prefix=/opt/python34
make
sudo make install
安装setuptool
tar axf  setuptools-26.1.1.tar.gz
cd setuptools-26.1.1
/opt/python34/bin/python3  setup.py build
sudo /opt/python34/bin/python3  setup.py install
安装pip
tar axf pip-8.1.2.tar.gz
cd pip-8.1.2
/opt/python34/bin/python3  setup.py build
sudo /opt/python34/bin/python3  setup.py install

配置系统环境： 在/opt/下面建一个bin目录，将要用的命令都链接过去

sudo mkdir /opt/bin
cd /opt/bin
ln -s /opt/python34/bin/python3 /opt/bin
ln -s /opt/python34/bin/pip3 /opt/bin

在配置到系统变量中

vim /etc/profile.d/opt.sh
export PATH=$PATH:/opt/bin
source /etc/profile.d/opt.sh
